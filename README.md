# JSON Kata

This is a personal exercise repository, where I learn programming languages by writing a json-parser following specification [FRC7159](https://tools.ietf.org/html/rfc7159) in a TDD approach.

## Erlang

- [x] Whitespace
- [x] Object
- [x] String
- [x] Escapes
- [x] Utf8
- [x] Utf16
- [x] Boolean and `null`
- [ ] Number
- [ ] Array
- [ ] Error position
